# React + Vite

Development - “Circular Economy Marketplace: Connecting Waste to Value” 

see the project - https://waste-to-value.vercel.app

 Build a Full-Stack Web/Mobile Application that: 
 
● Lists available recyclable materials (type, quantity, quality grade, 
location) from generators and recyclers 

● Matches suppliers with buyers using smart algorithms (price, 
location, material type, quality) 

● Tracks material provenance from collection to remanufacturing 
with blockchain-inspired transparency 

● Provides fair market pricing based on real-time demand and 
material specifications 

**tech stack** for the Waste-to-Value project:

## **Frontend Framework & Tools**
- **React** 19.2.0 - UI library
- **Vite** 7.2.4 - Fast build tool & dev server
- **React Router DOM** 7.13.0 - Client-side routing

## **Animation & UI**
- **Framer Motion** 12.29.2 - Smooth animations and transitions
- **Lucide React** 0.563.0 - Icon library

## **Styling**
- **CSS3** with custom properties (CSS Variables)
- Dark theme with neon emerald & gold accents
- Responsive design (mobile-first)

## **Development Tools**
- **ESLint** - Code linting
- **npm** - Package manager

## **Deployment**
- **Vercel** - Hosting & CI/CD (auto-deploys from GitHub)
- **GitHub** - Version control & repository

## **Architecture**
- **Context API** - State management (AuthContext, MarketplaceContext)
- **Component-based** - Reusable React components
- **Functional components** - Modern React patterns
